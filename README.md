## 專案簡介

本專案為參與2025 iThome鐵人賽，Software Development項下之參賽項目實作作品<br>
我在為期30天的Java自主學習中學習如何使用Java語法、Swing GUI製作遊戲畫面，並學習如何使用git與github儲存專案資料
