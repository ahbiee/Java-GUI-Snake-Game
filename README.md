## Build From Source 開發環境設置與執行
如果您是開發者，想要研究程式碼或進行修改，請參考以下步驟：

### Requirements 前置需求
- JDK 8 或以上版本
- Git

### Steps 執行步驟
1. **Clone 複製專案**
```bash
git clone https://github.com/ahbiee/Java-GUI-Snake-Game.git
cd Java-GUI-Snake-Game
```

2. **Compile 編譯程式碼**
```bash
# 假設原始碼在 src 資料夾中，或是直接位於根目錄
# 請依實際結構調整，例如：javac src/*.java
javac *.java
```

3. **Run 執行遊戲**

```bash
# 請將 Main 替換為您程式的主類別名稱
java Main
```
